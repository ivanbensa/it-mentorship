## JavaScript Exercises – switch

### Numbers + switch

1. Create a variable `number = 1`. Use switch to print “One”.
2. Create a variable `grade = 5`. Use switch to print “Excellent”.
3. Create a variable `day = 3`. Use switch to print “Wednesday”.
4. Create a variable `month = 12`. Use switch to print “December”.
5. Create a variable `statusCode = 2`. Use switch to print “Success”.

---

### String + switch

6. Create a variable `color = "red"`. Use switch to print “Stop”.
7. Create a variable `language = "JS"`. Use switch to print “JavaScript”.
8. Create a variable `role = "admin"`. Use switch to print “Admin access”.
9. Create a variable `city = "Belgrade"`. Use switch to print “Capital city”.
10. Create a variable `status = "online"`. Use switch to print “User active”.

---

### Boolean + switch

11. Create a variable `isOnline = true`. Use switch to print “Online”.
12. Create a variable `isAdult = false`. Use switch to print “Minor”.
13. Create a variable `hasAccess = true`. Use switch to print “Access granted”.

---

### Array + switch

14. Create an array `grades = [5, 4, 3]`. Use switch for first grade.
15. Create an array `statuses = ["online", "offline"]`. Use switch for second element.
16. Create an array `languages = ["HTML", "CSS", "JavaScript"]`. Use switch for third element.
