##JavaScript Exercises – switch

##Numbers + switch
1.	Create a variable number = 1.
Use switch to print “One” if the value is 1.
2.	Create a variable grade = 5.
Use switch to print “Excellent” if the grade is 5.
3.	Create a variable day = 3.
Use switch to print “Wednesday” if the value is 3.
4.	Create a variable month = 12.
Use switch to print “December” if the value is 12.
5.	Create a variable statusCode = 2.
Use switch to print “Success” if the code is 2.
