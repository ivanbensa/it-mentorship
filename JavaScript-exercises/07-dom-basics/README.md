## 07 Exercises – DOM (querySelector, getElementById, getElementsByClassName, innerText, innerHTML)

1. Create an HTML element `<p id="paragraph1">Hello</p>` and use `getElementById` to print its text to the console.  

2. Create an HTML element `<p class="text">JavaScript</p>` and use `getElementsByClassName` to print its text to the console.  

3. Create an HTML element `<h1 id="title">Title</h1>` and use `querySelector("#title")` to change the text to "New Title".  

4. Create HTML `<div class="box">Old content</div>` and use `querySelector(".box")` to change `innerText` to "New content".  

5. Create HTML `<p id="par1">Test</p>` and change `innerHTML` to `<strong>Bold Test</strong>`.  

6. Create two HTML elements `<p class="item">First</p>` and `<p class="item">Second</p>`. Change the text of the first element.  

7. Create HTML `<div id="main">Old div</div>` and add new text "Added" to the existing text.

9. Create HTML `<p class="para">Hello</p>` and change the text color to red.  

10. Create HTML `<div id="box">Old</div>` and insert `<p>New paragraph</p>` using `innerHTML`.  

11. Create HTML `<p id="text">Old text</p>` and add " + extra" to the existing text.  

12. Create HTML `<div class="card">Card 1</div><div class="card">Card 2</div>` and change both texts to "Changed".  

13. Create HTML `<span id="number">10</span>` and increase the number by 5 and print it to the console.
