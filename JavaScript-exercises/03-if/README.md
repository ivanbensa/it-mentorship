## Numbers and if

1. Create a variable `number = 10`.  
   If the number is greater than 5, print `"The number is greater than 5"`.

2. Create a variable `age = 18`.  
   If the age is equal to 18, print `"Adult"`.

3. Create variables `x = 3` and `y = 5`.  
   If `x` is less than `y`, print `"x is less than y"`.

4. Create a variable `points = 70`.  
   If `points` is greater than or equal to 60, print `"Passed"`.

5. Create a variable `temperature = 30`.  
   If the temperature is greater than 25, print `"It is hot"`.

## Boolean and if

6. Create a variable `isOnline = true`.  
   If `isOnline` is true, print `"User is online"`.

7. Create a variable `hasAccount = false`.  
   If `hasAccount` is false, print `"No account"`.

8. Create a variable `loggedIn = true`.  
   If `loggedIn` is true, print `"Successfully logged in"`.

9. Create a variable `isAdult = true`.  
   If `isAdult` is true, print `"Access allowed"`.

10. Create a variable `isAdmin = false`.  
    If `isAdmin` is false, print `"No admin rights"`.

## String and if

11. Create a variable `name = "Ana"`.  
    If the name is equal to `"Ana"`, print `"Hello Ana"`.

12. Create a variable `city = "Belgrade"`.  
    If the city is `"Belgrade"`, print `"Capital city"`.

13. Create a variable `language = "JavaScript"`.  
    If the language is `"JavaScript"`, print `"You are learning JS"`.

14. Create a variable `status = "online"`.  
    If the status is `"online"`, print `"User is active"`.

15. Create a variable `day = "Monday"`.  
    If the day is `"Monday"`, print `"Start of the week"`.

## Array and if

16. Create an array `numbers = [1, 2, 3]`.  
    If the first element is greater than 0, print `"Positive number"`.

17. Create an array `names = ["Ana", "Marko"]`.  
    If the first element is `"Ana"`, print `"Ana found"`.

18. Create an array `grades = [5, 4, 3]`.  
    If the first grade is equal to 5, print `"Excellent"`.

19. Create an array `statuses = [true, false]`.  
    If the second element is false, print `"Offline"`.

20. Create an array `languages = ["HTML", "CSS", "JavaScript"]`.  
    If the third element is `"JavaScript"`, print `"JS found"`.
